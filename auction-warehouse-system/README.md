## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open http://localhost:5173

## Deployment
This project is configured for GitHub Pages deployment.
